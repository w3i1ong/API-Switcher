# Claude API Switcher — 反馈与功能建议

本仓库是 **Claude API Switcher** 的公开问题追踪页面，这是一款用于在 VS Code 中切换 Claude Code API 配置的插件。

> ⚠️ 本仓库**不包含源代码**，仅用于收集 Bug 反馈和功能建议。

---

## 🐛 报告 Bug

提交前请：

1. 搜索 [已有 Issue](../../issues)，避免重复提交
2. 确认你使用的是最新版本的插件

[**→ 提交 Bug 报告**](../../issues/new?template=bug_report.md)

---

## 💡 功能建议

有什么想法？欢迎告诉我们。

[**→ 提交功能建议**](../../issues/new?template=feature_request.md)

## 📋 提交规范

为了帮助我们更快定位问题，请尽量提供：

- **插件版本**（在 VS Code 扩展面板中查看）
- **VS Code 版本**（`帮助 > 关于`）
- **操作系统**
- Bug 报告请附上：复现步骤、预期行为与实际行为
- 欢迎附上截图或录屏
